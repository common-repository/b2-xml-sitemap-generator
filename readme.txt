== Description ==

Sitemap Generator.

Some features:
* Generates XML and HTML sitemaps automatically when changes occur
* Set priorities
* Set Frequency
* Include/Exclude categories from sitemaps
* Pings Google/Bing/Ask.com automically about sitemap changes

== Installation ==

1. Upload the `b2-xml-sitemap-generator` folder to the `/wp-content/plugins/` directory
2. Activate the B2 XML Sitemap Generator plugin through the 'Plugins' menu in WordPress
3. Configure the plugin by going to the `B2 XML Sitemap Generator` menu that appears in your admin menu
